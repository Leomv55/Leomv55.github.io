ARM Migration + OpenSearch

Migrated OpenSearch from x86 to ARM — benchmarked before migrating, saved $11,000/year.

$11k/year saved
AWS OpenSearch ECS GitHub Actions Python

Problem

OpenSearch cluster running on x86 — suspected ARM would be cheaper but no data to back it.

Approach

Built a benchmarking script comparing x86 vs ARM on the actual workload. ARM came out ahead. Migrated ECS nodes, updated to ECS-optimised ARM AMIs. Along the way: fixed a 7GB memory issue from an _id-field sorting anti-pattern, migrated CI/CD from Jenkins to GitHub Actions.

Outcome

$11,000/year saved. Currently also leading solo migration from OpenSearch 2 → OpenSearch 3 (derived vector feature saves 40% storage).

Stack

AWS ECS · OpenSearch · CloudFormation · GitHub Actions · Python
